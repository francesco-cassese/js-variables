# JS Variables

## 📌Descrizione

Esercizio fondamentale per l'apprendimento della logica di programmazione in JavaScript, focalizzato sulla gestione dei dati e sull'utilizzo degli strumenti di debugging del browser.

---

## 🎯 Obiettivo

L'obiettivo principale è padroneggiare i concetti base della manipolazione dei dati in JavaScript, passando dalla teoria alla pratica attraverso una serie di task incrementali.

### Focus Tecnico:

* **Data Types:** Manipolazione di diversi tipi di dati (Stringhe, Numeri, Booleani) e comprensione della loro natura.
* **Debugging:** Utilizzo intensivo dei **Chrome DevTools** e di `console.log()` per ispezionare il comportamento del codice in tempo reale.
* **Logica Applicata:** Risoluzione di piccoli problemi algoritmici contenuti negli script forniti.

---

## 🛠️ Tecnologie Utilizzate

* **HTML5:** Struttura di base per il caricamento degli script.
* **JavaScript (ES6+):** Linguaggio principale per lo svolgimento delle task.
